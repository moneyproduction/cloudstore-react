Доброго дня, це мій показовий проект CLOUDSTORE на ReactJS, в ньому задіяні технології Redux Toolkit, React Router v6, Axios + Fetch, React Hooks, React Content Loader, React Pagination, Lodash. Debounce. Як перейдете за посиланням на проект, просто клацніть на іконку/назву в шкапці сторінці, щоб перейти на головну, бо github трохи криво завантажив. Як Back-end я беру сервіс mockapi. Над красою та адаптивом сайту не заморочувався, головне було показати функціональність сайту. Дякую за увагу.
