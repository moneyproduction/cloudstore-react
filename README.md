Hello, this is my demonstration project CLOUDSTORE on ReactJS, it uses technologies Redux Toolkit, React Router v6, Axios + Fetch, React Hooks, React Content Loader, React Pagination, Lodash. Debounce. When you go to the link to the project, just click on the icon/name in the header of the page to go to the main page, because GitHub loaded a little crookedly. I use the mockapi service as a back-end. Idid not bother about the beauty and adaptability of the site, the main thing was to show the functionality of the site. Thank you for attention.
